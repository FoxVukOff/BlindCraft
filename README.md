# BlindCraft: Эхо Глубин — Finale Edition

Текстовый мобильный квест о слепом рудокопе. 5 актов, от строительства дома до победы над Драконом Края.

## ⚠️ Проект завершён

Больше никаких обновлений, исправлений или поддержки не будет. Игра остаётся доступной и playable.

- 📧 Почта: foxvuksautoreplysystem@gmail.com
- 🐱 Новый проект: [«Котики»](https://g115r51mjrc0-d.space-z.ai/)

## Технологии

- **Next.js 16** (App Router, Turbopack)
- **TypeScript 5**
- **Tailwind CSS 4** + shadcn/ui
- **Prisma ORM** (SQLite)
- **Web Audio API** — синтезированный звук
- **localStorage** — сохранения

## Установка

```bash
bun install
bun run dev
```

Откройте `http://localhost:3000`

## Деплой на Vercel

1. Загрузите репозиторий на GitHub
2. Импортируйте в Vercel
3. Vercel автоматически определит Next.js
4. Деплой

## Структура

```
src/
  app/           — Next.js App Router
  components/    — React-компоненты игры
    game/        — игровые компоненты
  hooks/         — React hooks
  lib/
    game/        — игровой движок, сюжет, звук, сохранения
```
